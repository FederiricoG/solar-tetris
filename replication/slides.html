// Foundation

ML vs. LLMs & The Physics of Hallucinations

01/ Traditional ML vs. LLMs

Machine Learning (ML): Specialized mathematical models trained on specific datasets for single-focus tasks (e.g., concrete crack detection, cost prediction).

Large Language Models (LLMs): General cognitive engines. Trained on semantics to write, reason, generate code, and synthesize complex multi-domain ideas out-of-the-box.

02/ Why LLMs Hallucinate

LLMs do not query a database of static truths. They are probabilistic token predictors—they predict the next most logical word based on context patterns.

The Cure: Grounding. By feeding the model verbatim source texts (e.g., BayBO building codes) and enforcing structural constraints, we bind its reasoning to reality.

// Prompt Engineering

Rules of Agentic Prompting & Verification

STEP 01

Structured Output

Design prompts to generate predictable, machine-readable structures (e.g., JSON schemas or markdown tables) instead of conversational paragraphs.

STEP 02

Separation of Concerns

Never ask a single prompt to do everything. Stack models dynamically: one for extracting facts, one for drafting layouts, one for describing experience.

STEP 03

Empirical Loops

Treat AI output as testable hypotheses. Create feedback loops: Plan -> Execute -> Benchmark -> Refine to continuously trace and catch error loops.

// Systems Thinking

Folders & Markdown: The Filesystem as Architecture

Case Study: Interpreted Context Methodology (ICM)

Open-source framework by Jake van Clief (RinDig/Interpreted-Context-Methdology)

1. Folders as Execution Stages

Organize pipelines into numbered folders (e.g., 01_analysis/). AI processes only the context files within that scope, minimizing noise.

2. Markdown (.md) as State Storage

Plain-text markdown is lightweight, easily readable by LLMs, and formats headers, lists, and tables perfectly for context parsing.

3. Human-in-the-Loop

Because states are stored on disk as files rather than database variables, developers can inspect, edit, or override any step instantly.
